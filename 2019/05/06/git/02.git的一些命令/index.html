02.git的一些命令

发表于 2019-05-06 | 分类于 git

参考，个人作为记录而已
廖雪峰git教程

初始化一个Git仓库：git init

在一个目录A里面，右键Git Bash Here，在打开的界面运用这个命令，git init命令可以把这个目录变成Git可以管理的仓库，就是说可以管理版本控制了

通过什么管理？init之后，当前目录下多了一个.git的目录，这个目录是Git来跟踪管理版本库的

添加文件到Git仓库：分两步

  1. 使用命令git add <file>，注意，可反复多次使用，添加多个文件；

    就是说，可以git add a，git add b。。。，提交呢，可以一次性将所有add的文件提交

    第四篇会说到，其实是add到了暂存区

  2. 使用命令git commit -m <message>，完成。

为什么Git添加文件需要add，commit一共两步呢？因为commit可以一次提交很多文件

其实修改文件也是这两步，就是这两条命令
